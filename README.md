## Shortcuts

Shortcut | Action
---------| --------
modkey + Enter | xterm
modkey + r | run prompt
modkey + F1 | Go to main screen
modkey + F2 | Go to second screen
modkey + o | Send window for other screen
Ctrl + Shift + l | slock
modkey + f | Fullscreen
